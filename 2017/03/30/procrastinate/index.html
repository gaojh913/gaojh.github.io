拖延症患者的自我拯救

发表于 2017-03-30 | 分类于 成长/心情
简介

    计划已久的个人博客终于在延期半年后的今天搭建完成了，由此可见拖延已经成为我在自我成长过程中的极大障碍。因此，在阅读过《拖延症心理学》一书后，我整理了拖延症的特点及危害、产生根源、以及处理拖延的技巧。并在此基础上制订了一份适合自己的自我拯救计划，以求摆脱拖延的困扰。同时也希望以下文字可以为有着相同困扰的人们提供一些帮助。

拖延的特点及危害
  • 拖延有其自身的生命和意志，不会被彻底消灭
  • 拖延会引发内在的痛苦，也会产生严重的外在后果
产生拖延的根源及因素
  • 根源
    • 家庭背景、儿时环境产生拖延
    • 生理（不断变化的大脑，意识，记忆）产生拖延
    • 利用拖延逃避令自己不适的感受，来实现自我保护
    • 人际关系、社会关系加剧拖延，从而获得一个忠实的伙伴
    • 对时间的期望偏差导致客观时间与主观时间冲突，会产生、加剧拖延
  • 因素
    • 无法有效的自我约束
    • 对成功所需的能力缺乏自信
    • 对要去完成某个任务有反感心理
    • 目标和回报太遥远了，感受不到意义
拖延的处理技巧
  • 目标
    • 确立一个可操作的目标
    • 设定一个务实的目标
    • 将你的目标分解成短小具体的迷你目标
    • 按照重要程度对目标进行排序
    • 集中精神完成目标
    • 避免同时进行多个目标
    • 建立奖励机制，鼓励自己完成目标
  • 时间
    • 现实地对待时间
    • 从现在开始
    • 以能保持专注为标准分割时间
    • 为自己设下时间限制
    • 保护时间，不要去做额外的或者不必要的事情
    • 将最喜欢的事放在最后
  • 其他
    • 不利用借口来拖延，而是将其视为继续的信号或阶段完成后的自我奖励
    • 将拖延看成是一个信号，并分析其传递的信息
    • 正确面对拖延。不要试图消灭拖延，而是接受适应并利用拖延
自我拯救计划
  1. 每天做一次总结，整理收获并形成一篇博文
  2. 重新刷一次FCC，每天有效时间不低于三小时
  3. 重新梳理ES6新特性，每天有效时间不低于一小时
  4. 每两天手写至少一个前端特效并上传github
  5. 复习巩固node.js并尝试用其作为后端建立独立个人博客
  6. 重新开始学习日语，每天有效时间不低于一小时
